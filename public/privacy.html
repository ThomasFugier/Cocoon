Politique de confidentialité

WeSpice est une application privée pour couple. Cette page explique quelles données sont utilisées, pourquoi elles le sont, et comment demander leur suppression.

Données traitées

  • Compte : identifiant Supabase, nom affiché, email fourni par Google ou Apple, fournisseur de connexion.
  • Espace couple : code d'invitation, membres, statut du jour, préférences de notifications.
  • Envies et matchs : votes, cartes personnalisées, matchs révélés par chaque utilisateur.
  • Chat privé : messages texte, photos envoyées, dates d'expiration et état de consommation des photos à vue unique.
  • Achats : identifiants de produits, entitlements et statut de validation RevenueCat.
  • Notifications : token Expo Push, plateforme et préférences activées.

Utilisation

Ces données servent uniquement à faire fonctionner l'application : synchroniser le couple, cacher ou révéler les matchs selon les règles, envoyer les messages privés, valider les achats, restaurer les accès et envoyer les notifications demandées.

Conservation

Les messages privés expirent automatiquement autour de 06:00, heure de Paris. Les photos reçues sont floutées, visibles une seule fois pendant une courte durée, puis supprimées côté serveur. Les autres données restent conservées tant que le compte existe, sauf suppression demandée par l'utilisateur.

Services tiers

WeSpice utilise Supabase pour l'authentification, la base de données, le stockage privé et les Edge Functions, Expo pour les notifications push, RevenueCat pour la validation des achats, ainsi que Google, Apple et Google Play/App Store pour l'authentification et les achats natifs.

Suppression du compte et des données

La suppression est disponible dans l'application : Profil > Application > Supprimer mon compte. Elle supprime le compte serveur, le profil, les votes, les messages, les photos privées et les tokens de notification associés à l'utilisateur. Si un partenaire reste dans le couple, son compte et ses données personnelles ne sont pas supprimés.

Une page dédiée décrit aussi ce parcours : suppression compte et données.

Contact

Pour une demande liée aux données personnelles, contactez l'éditeur de WeSpice à l'adresse de support indiquée sur la fiche App Store ou Google Play. Adresse de support prévue : support@wespice.app.
